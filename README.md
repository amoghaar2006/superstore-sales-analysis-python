# Superstore Sales Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Superstore retail dataset using Python. The objective is to analyze sales performance, profit trends, customer purchasing patterns, and regional performance using data visualization and aggregation techniques.

---

## Objectives

- Understand the structure of the dataset
- Clean and preprocess the data
- Perform exploratory data analysis
- Identify sales and profit trends
- Generate meaningful business insights using visualizations

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

---

## Dataset Information

The dataset contains retail sales transactions including:

- Order Details
- Customer Information
- Product Categories
- Sales
- Profit
- Discount
- Region
- State
- Order Date
- Ship Date

---

## Data Preprocessing

The following preprocessing steps were performed:

- Loaded the dataset
- Checked missing values
- Checked duplicate records
- Verified data types
- Converted date columns into datetime format
- Extracted Year and Month from Order Date and Ship Date

---

## Exploratory Data Analysis

The project includes the following analyses:

- Sales by Year
- Category-wise Sales
- Region-wise Sales
- Top 10 States by Sales
- Top 10 Products by Sales
- Category-wise Profit
- Sales Distribution using Pie Chart
- Sales vs Profit Scatter Plot

---

## Key Insights

- Technology generated the highest sales.
- The West region recorded the highest overall sales.
- California contributed the highest sales among all states.
- Office Supplies received the highest number of orders.
- A few products contributed a significant portion of total revenue.
- Higher sales generally resulted in higher profits, although some high-sales transactions resulted in losses.
